# Compser Landing Page

A beautifully responsive, high-fidelity dark mode landing page built with **Next.js**, **React**, and **Tailwind CSS**. 

## 🖼️ Final Design Snapshot

Here is the exact pixel-perfect implementation of the required design grid and layout:

![Full Landing Page Screenshot](./public/fullscreenshot.png)

## 🚀 Features
- **Responsive Layout**: Adheres to a strict component lifecycle and grid boundaries across all viewports (`px-4`, `md:px-8`, `max-w-[1500px]`).
- **Tailwind v4 Integration**: Uses the newest arbitrary aspect ratios (`aspect-4/5`, `aspect-4/3`) elegantly.
- **Dynamic Assembling**: Uses continuous abstract layouts, overlapping components, and interactive micro-animations (e.g. grayscale to color hover reveals).
- **Dark UI Theme**: Sophisticated `#050505` backgrounds with `#f6f6f6` typography built around the Inter font family.

## 🛠️ Stack
- [Next.js](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) for iconography.
